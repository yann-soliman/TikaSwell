# Product Scope

## MVP In

- Display current conditions for the configured initial spot
- Display an estimated surf score based on historical similarity
- Enter a surf session with:
  - date
  - start time
  - end time
  - rating out of 10
  - optional notes
- Show historical sessions
- Allow session editing later in the same app
- Persist condition snapshots locally

## MVP Out

- Multi-user support
- Multi-spot support
- Email alerts
- Gotify alerts
- Third-party forecast tool integrations
- Public API
- Mobile app
- Advanced machine learning

## Future Scope

- Multiple providers
- Multiple spots
- Provider comparison
- Alert rules for good conditions
- Better scoring confidence
